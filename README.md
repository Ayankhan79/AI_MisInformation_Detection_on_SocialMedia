🤖 **AI Misinformation Detection**

Built a machine learning pipeline to classify **real vs. misinformation** social media posts using a synthetic dataset of **5,000 records**.

**Key Contributions:**

* Cleaned and preprocessed the data, encoded categorical features, and handled missing values.
* Engineered features such as **Engagement Rate**, **Viral Ratio**, and **Account Authenticity**.
* Compared **Logistic Regression, Random Forest, and XGBoost** using Accuracy and ROC-AUC.

**Results:**

* Achieved **~91.6–91.7% accuracy** across models.
* Found that **Logistic Regression matched XGBoost's performance**, highlighting the impact of strong feature engineering.
* Identified **account authenticity and bot-related metrics** as the strongest predictors of misinformation.

This project strengthened my skills in **feature engineering, classification modeling, model evaluation, and explainable machine learning**.
